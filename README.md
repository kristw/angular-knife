angular-knife
=============

Lightweight and useful services, filters and directives for angularjs

---

Install via bower
```
bower install angular-knife
```

Then include `ngKnife` module in your angular app.

---

Services
- defer
- debounce
- safeApply
 
Filters
- iif
- positiveNegative
- sign
- truncate
 
Directives
- stopEvent
